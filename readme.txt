== Description ==

DD Favourite Posts enable wordpress users to display list of favourite/highlighted posts on the sidebar. Sidebar widget is available.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `DDFavouritePosts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select your favourite posts in Dashboard> Settings> DD Favourite Posts
4. Get your widget at Dashboard> Appearance> Widget
